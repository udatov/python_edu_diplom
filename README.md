# Проектная работа: диплом

## Про удачу

У вас будет один репозиторий на все 4 недели работы над дипломным проектом. 

Если вы выбрали работу в командах, ревью будет организовано как в командных модулях с той лишь разницей, что формируете состав команды и назначаете тимлида вы сами, а не команда сопровождения.

Удачи!

## Подготовка среды
Запуск приложения требует **минимальной** среды: **Docker** (тестировалось на версии **27.4.0**); 

Для подготовки коллекции базовых Docker-имиджей запустите команду:

`make build_all-threatre-base_containers`

Данная команда соберет базовый Docker-имидж для каждого прикладного сервиса (например, **auth_api_base**), тем самым мы минимизируем время сборки имиджей **запуска**
сервиса (например, **auth_api**): базовые Docker-имиджи собираются только в том случае, если изменилися состав python пакетов прикладного сервиса (например, вышла новая версия FastAPI) или делаем инфраструктурные изменения, например, меняем имя пользователя, запускающего entry-point скрипт сервиса. Поэтому запуск данную команды должен проходить редко. 

Для подготовки среды **разработки** потребуется:

**python3 (ver >= 12)**; **pip**; менеджер проектов [uv](https://docs.astral.sh/uv/).

Далее запустите команду: `make prepare-dev-env`

## Запуск сервиса

`make run_gui_docker`

Стартовая страница должна быть доступна по адресу: `http://localhost:8008/login`
